# TypeItAlready

TypeItAlready is a typing game for computer lovers and geeks to improve their typing speed.

It is developed and written completely in Java. The application is suitable for desktops.

Java packages used :

      java.awt
      java.swing
      java.io
      java.util
      
 
Java subpackages used :
      java.awt.event
      java.swing.event
      java.swing.text
